Hospital Full Stack Application

Frontend: React
Backend: FastAPI
Database: MySQL
IDE: Visual Studio Code

Setup Details: 

1. Create 2 folders: frontend and backend
2. In the backend folder, use it as an API application 
3. Install the necessary packages, can refer backend-api-requirements.txt
4. Use the fronted folder for react appllication
5. Then open a terminal and install npm package
6. Wait for packages to download and the file extensions created
7. Create a folder named page in src and add the required javascript codes
8. Then, in app.js, mention the required routes
9. Open a terminal and run the server by typing npm start
10. Then the React page opens in a browser, displaying the UI
11. To stop the server, press Ctrl + C

Features added:

1. React Application
2. Frontend and backend connection establishment
3. Web Sockets with FastAPI
4. Pagination for APIs
5. Search & filtering AIs
6. File upload and download api
7. Login UI and Dashboard UI
8. Doctor listing in the dashboard
9. Patient listing in the dashboard
10. Appointment booking feature in dashboard
11. Appointment Notification features
12. Testing application using Pytest
